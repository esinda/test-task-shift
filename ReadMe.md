## Инструкция по запуску утилиты

### Описание

Утилита фильтрует содержимое входных файлов по типам данных (целые числа, числа с плавающей точкой, строки) и записывает их в отдельные файлы.

### Поддерживаемые опции

- `-o <path>` — путь для выходных файлов (по умолчанию текущая папка)
- `-p <prefix>` — префикс для имён выходных файлов (по умолчанию пустой)
- `-a` — добавление к существующим файлам, иначе файлы перезаписываются
- `-s` — краткая статистика (количество элементов)
- `-f` — полная статистика (минимум, максимум, сумма и среднее для чисел; длина строк)

### Требования

- Java 17
- Maven 3.14.1

### Сборка

В корне проекта выполните:

```bash
mvn clean package
```

### Запуск

С использованием входных файлов in1.txt и in2.txt:

```bash
java -jar target/test-task-shift-1.0.0.jar -s -f -a -p sample- src/main/resources/in1.txt src/main/resources/in2.txt
```

### Результат

Файлы с данными:

sample-integers.txt

sample-floats.txt

sample-strings.txt

### Примечания

- Если для какого-то типа данных нет элементов, файл не создаётся.

-Код написан без сторонних библиотек, используется только стандартная Java.
